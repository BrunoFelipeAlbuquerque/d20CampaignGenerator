# Solution Notes

This directory stores reusable project learning from completed work.

Use it only after review, when a task reveals a project-specific rule, boundary, or misuse pattern that should help future tasks.

Notes here do not create tasks and do not override `ISSUES.md` or `BACKLOG.md`.
