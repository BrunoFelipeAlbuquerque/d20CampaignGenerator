Pathfinder Unchained: Fractional Base Bonuses

Source:
- Pathfinder Unchained, pages 40-41.
- Reference URL: https://aonprd.com/Rules.aspx?ID=1726
- Mirror URL: https://www.d20pfsrd.com/gamemastering/other-rules/unchained-rules/fractional-base-bonuses/

Project use:
This repository models core class base attack bonus and base save progressions with the Pathfinder Unchained fractional base bonuses variant.

Rule summary:
- Calculate each class contribution as a fraction.
- Add all class contributions together before rounding down to the game-facing value.
- Base attack bonus progressions:
  - d6 Hit Die classes add 1/2 per class level.
  - d8 Hit Die classes add 3/4 per class level.
  - d10 and d12 Hit Die classes add 1 per class level.
- Base save progressions:
  - good saves add 1/2 per class level.
  - poor saves add 1/3 per class level.
- For each save type, add the good-save base bonus of +2 once if at least one class has a good progression for that save.
- The good-save +2 does not stack between classes under this variant.
- A class has a good save progression when its 1st-level core table bonus for that save is +2.
- A class has a poor save progression when its 1st-level core table bonus for that save is +0.
- Iterative attacks still use the total rounded base attack bonus thresholds: +6, +11, and +16.

Examples:
- Fighter 1 plus barbarian 1 Fortitude: 1/2 + 1/2 + 2 = 3, not 4.
- Cleric 5 plus fighter 2 Fortitude: 5/2 + 1 + 2 = 11/2, rounded down to 5.
- Cleric 5 plus fighter 2 Reflex: 5/3 + 2/3 = 7/3, rounded down to 2.
- Cleric 5 plus fighter 2 Will: 5/2 + 2/3 + 2 = 31/6, rounded down to 5.
